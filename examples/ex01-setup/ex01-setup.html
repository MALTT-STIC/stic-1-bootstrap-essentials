Bootstrap basic setup

For an HTML page to take advantage of the Bootstrap framework you need:

  1. To import the CSS file bootstrap.css (or bootstrap.min.css).
    <link href="/path/to/css/bootstrap.min.css" rel="stylesheet">
  2. (Optional) To import the CSS file bootstrap-theme.css (or bootstrap-theme.min.css).
    <link href="/path/to/css/bootstrap-theme.min.css" rel="stylesheet">
  3. To include the jQuery JavaScript library.
    <script src="/path/to/jquery.min.js"></script>
  4. To include the bootstrap.js (or bootstrap.min.js) file.
    <script src="/path/to/js/bootstrap.min.js"></script>
  5. To define the viewport meta tag in the head.
    <meta name="viewport" content="width=device-width, initial-scale=1">
Remember: you can use Bootstrap only with an HTML5 page!
